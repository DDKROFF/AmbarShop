# AmbarShop

AmbarShop — это веб-приложение интернет-магазина зоотоваров с каталогом товаров, корзиной, личным кабинетом, доставкой, оплатой, статьями по уходу за животными и системой отзывов.

## 1. Назначение проекта

Проект предназначен для организации онлайн-продаж зоотоваров с возможностью:
- просмотра каталога товаров и карточек товаров;
- добавления товаров в корзину и оформления заказа;
- выбора способа доставки и оплаты;
- управления адресами доставки и личным профилем;
- просмотра статей и гайдов по уходу за животными;
- оставления отзывов и отправки сообщений через форму обратной связи.

## 2. Техническое задание

### 2.1 Цели проекта
- обеспечить удобный и понятный процесс покупки зоотоваров онлайн;
- реализовать полный цикл заказа: от выбора товара до оплаты и доставки;
- предоставить пользователю личный кабинет для управления персональными данными и заказами;
- внедрить информационный раздел с гайдами и статьями по уходу за животными;
- обеспечить возможность работы через веб-интерфейс и REST API.

### 2.2 Основные функции
- каталог товаров с категориями, поиском и сортировкой;
- карточка товара с изображениями, описанием, ценой и остатком на складе;
- корзина и оформление заказа;
- личный кабинет пользователя с профилем, адресами доставки, заказами и настройками;
- интеграция с Яндекс.Картами для выбора и отображения адреса доставки;
- система оплаты с поддержкой статусов платежа и чеков;
- раздел с отзывами и статьями;
- форма обратной связи.

### 2.3 Требования к системе
- адаптивный интерфейс для мобильных и десктопных устройств;
- безопасная регистрация и авторизация пользователей;
- хранение персональных данных и адресов доставки;
- поддержка REST API для работы с каталогом, заказами, доставкой, оплатой и контентом;
- поддержка изображений товаров и аватаров пользователей.

### 2.4 Технологический стек
- Python 3.14
- Django 6.0.6
- Django REST Framework 3.17.1
- PostgreSQL
- HTML, CSS, JavaScript
- Pillow
- Poetry

## 4. Архитектура проекта

Проект построен по модульной архитектуре с разделением на приложения:
- core — служебная информация и базовые страницы сайта;
- shop — товары, категории, каталог и карточки товаров;
- guides — статьи, гайды и взаимодействие пользователей с контентом;
- delivery — способы доставки, адреса и корзина;
- payments — способы оплаты, платежи и чеки;
- account — пользователи, профили, отзывы, корзина, заказы и личный кабинет.

## 5. Структура проекта

- account — регистрация, авторизация, профиль, заказы, адреса, отзывы;
- shop — модели категорий, товаров и изображений;
- guides — статьи и статьи с лайками/дизлайками;
- delivery — методы доставки, адреса и корзина;
- payments — платежи, способы оплаты и чеки;
- templates — HTML-шаблоны страниц;
- static — CSS, JavaScript и изображения.

## 6. Установка и запуск

1. Установите зависимости:
```bash
poetry install
```

2. Выполните миграции:
```bash
poetry run python manage.py migrate
```

3. Создайте суперпользователя:
```bash
poetry run python manage.py createsuperuser
```

4. Запустите сервер:
```bash
poetry run python manage.py runserver
```

## 7. Яндекс.Карты

Для интеграции карт добавьте API ключ:
1. Получите бесплатный API ключ на [Яндекс.Конструктор](https://developer.tech.yandex.ru/)
2. Откройте templates/account/map.html
3. Замените YOUR_API_KEY на ваш действительный ключ
4. Перезапустите сервер

## 8. API endpoints

Все API доступны по адресу /api/:
- /api/categories/ — категории товаров;
- /api/products/ — товары;
- /api/guides/ — гайды;
- /api/delivery-methods/ — способы доставки;
- /api/delivery-addresses/ — адреса доставки;
- /api/orders/ — заказы;
- /api/payment-methods/ — способы оплаты;
- /api/payments/ — платежи;
- /api/receipts/ — чеки;
- /api/site-info/ — служебная информация.
